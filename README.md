FinSight.AI — Product Description

Overview

FinSight.AI is an interactive personal finance analytics dashboard that combines AI-powered transaction categorization with real-time cash-flow visualization. Designed for individuals who want deeper insight into their spending habits, it transforms messy bank logs into structured, actionable financial data — all inside a sleek, glassmorphism-style interface.

---

Key Features

Feature | Description |

🧠 AI Auto-Categorizer: Type any raw transaction string (e.g. *"Swiggy biryani 350"*) and a simulated LLM pipeline classifies it into Food, Travel, Utilities, Shopping, or Subscriptions — in real time. 
📊 Live Spending Charts: A dynamic doughnut chart breaks down your category-wise outflow, while a trend line chart plots 6-month spending history alongside an AI-generated forecast for the upcoming month. 
📈 **Smart KPI Metrics: At a glance, track your Total Outflow, Savings Rate, AI Classification Accuracy, and Projected Monthly Spend — all recalculated dynamically as you add transactions. 
🔍 Filterable Transaction History: Browse and filter your full transaction log by category using quick-access pill buttons. Every new entry animates into the table instantly. 
📥 One-Click CSV Export: Download your full transaction history as a formatted `.csv` report for use in spreadsheets or accounting tools. 
✨ Liquid Glass UI: Cards respond to cursor movement with 3D tilt and specular lighting effects, creating a premium, tactile experience. 


Who Is It For?

FinSight.AI is built for developers, designers, and product teams who want to see how AI-assisted finance tools can work — and for individuals looking for a modern, interactive way to track and understand their personal spending.
